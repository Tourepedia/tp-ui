# Badge component

Use it to show `state` or `count` into the UI

## Installation

```bash
yarn add @tourepedia/badge
```

## Usage

```js
// import the badge component
import Badge from "@tourepedia/badge"

const MyComponent = () => {
  return (
    <div>
      Notifications <Badge>10</Badge>
    </div>
  )
}
```
