### Basic Usage

Use `.btn` class name to apply a btn look to any component.

```html live=true
<button class="btn" type="button">Click Me!</button>
```

### Variants

Use `.btn-primary`, `.btn-secondary` and `.btn-tertiary` to add a hierarchy to the button element. Use `.branded`,
`.success`, `.danger`, `.warning` to add a visual semantics to the button element

## Playground

```html live=true
<button class="btn btn-primary branded">Update</button>
<button class="btn warning">Delete</button>
```
