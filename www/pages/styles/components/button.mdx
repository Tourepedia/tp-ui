### Basic Usage

Use `.btn` class name to apply a btn look to any component.

```html live=true
<button class="btn" type="button">Click Me!</button>
```

### Variants

Use `.btn-primary` to apply the primary color styles to the button

```html live=true
<button class="btn btn-primary" type="button">Click Me!</button>
```
