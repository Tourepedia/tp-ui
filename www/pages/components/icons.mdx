## Available Icons

Bellow is the list of available icons. Each icon has a label which represents the `import` name for that icon.

```jsx preview=true
function () {
  return <div>
  {Object.keys(Icons).map(key => {
    if (!Icons.hasOwnProperty(key)) return null
    const Icon = Icons[key]
    if (typeof Icon !== "object") return null
    return <div key={key} className="inline-block w-32 my-4 text-center">
      <span
        className="w-12 h-12 border rounded inline-flex justify-center items-center">
        <Icon />
      </span>
      <br />
      <div className="text-xs mt-2 tracking-wider text-gray-700 hover:text-gray-900">
        {Icon.displayName}
      </div>
    </div>
  })}
  </div>
}
```

## Props

All the icons are SVG components and so accepts all the SVG props. Along with SVG props, following props are accepted
as well:

#### title
type: string

To make icons accessible when there is not text associated with the icon, you can provide a title prop to the icons.

```jsx live=true
<Button primary branded>
  <Icons.Cancel title="Cancel the procces" />
</Button>
```

#### rotate
oneOf: "90" | "180" | "270"

You can rotate icons by passing a `rotate` prop for some use cases where icon is provided for only one direction e.g.
`Icons.ChevronDown`


```jsx live=true
<Button primary branded>
  Next <Icons.ChevronDown rotate="270" />
</Button>
```
